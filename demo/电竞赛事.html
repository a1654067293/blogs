1 简介

对电竞赛事信息的信息展示，包括首页信息、赛事中心和各个电竞比赛专区。对英雄联盟专区实现赛程日历，比赛阶段排名、参赛战队的详细信息以及各个战队比赛的对阵图。

2 主要内容

  1. 轮播图展示：使用 Element Plus 的 Carousel 组件实现轮播图功能，并通过 :deep() 深度选择器获取指示器 DOM 元素，实现小图控制大图的效果。
  2. 阶段排名：通过双重 Tabs 组件展示阶段排名，使用 Table 组件渲染战队排名数据，并利用插槽解决在表格中添加图片的问题。
  3. 路由跳转：使用 Menu 组件和子路由实现路由跳转，并在需要显示的页面使用 router-vew。
  4. 视频懒加载：通过分页功能实现精彩视频的懒加载，提高用户体验。
  5. 数据展示：通过 Axios 从 API 获取比赛数据，展示在对阵图中，包括比赛轮次、战队对战情况、比分和时间。根据战队比分动态调整样式以区分胜负。
  6. 国际化语言切换：通过使用vue3的Config Provider 全局配置组件，实现语言的切换功能。
  7. 弱网环境性能优化：通过使用图片懒加载、CDN加速和压缩静态资源等方法来减少请求耗时，添加加载动画提高用户体验。
